# CHIP8 Emulator/Interpreter

- CHIP8 Emulator written in C with SDL library developed in Linux & Windows

![image](https://github.com/ajarred/chip8/assets/69410244/cee63c52-7af7-482d-b22a-3bcb718f48c1)

### Building
```
make
```

### Usage
```
./chip8 <rom_file_path>
```
